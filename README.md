# Визуализатор топологической сортировки

Учебная практика «Новые языки программирования», кафедра МОЭВМ СПбГЭТУ «ЛЭТИ», 2026.

## Описание

Финальная версия приложения визуализирует алгоритм Кана для топологической сортировки ориентированного графа. Граф можно построить в интерфейсе или загрузить из текстового файла.

## Возможности Release 1.0

- создание вершин и ориентированных рёбер;
- перемещение ребра вместе с его конечными вершинами;
- удаление вершин и рёбер;
- загрузка и сохранение графа;
- ручное пошаговое выполнение алгоритма;
- шаг назад;
- автоматическое выполнение с настраиваемым интервалом;
- остановка автоматического режима;
- отображение очереди, входящих степеней и текущего результата;
- цветовая подсветка текущей вершины и текущего ребра;
- обнаружение циклов;
- сохранение результата в отдельный файл;
- обработка пустых, повреждённых, слишком больших и неподходящих файлов;
- прокрутка рабочей области для больших графов;
- окно «О разработчиках».

## Запуск

Требуется Java 17 или новее.

```bash
java -jar releases/toposort-visualizer-release.jar
```

## Сборка

Windows:

```bat
build.bat
```

Linux/macOS:

```bash
sh build.sh
```

## Формат входного файла

```text
VERTICES 5
0 100 100
1 300 100
2 100 260
3 320 260
4 520 180
EDGES 5
0 1
0 2
1 3
2 3
3 4
```

Пустые строки и строки-комментарии, начинающиеся с `#`, игнорируются.

## Структура

```text
src/        — исходный код
samples/    — корректные и некорректные примеры входных файлов
docs/       — спецификация
tests/      — описание проверок
releases/   — готовый JAR
results/    — место для сохранённых результатов
```
