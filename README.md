# Визуализатор топологической сортировки

Учебная практика на кафедре МОЭВМ ЛЭТИ, 2026.

## Тема проекта

Разработка визуализатора алгоритма топологической сортировки на языке Java с графическим интерфейсом.

## Описание проекта

Приложение предназначено для визуализации алгоритма топологической сортировки ориентированного графа.

Пользователь сможет загрузить граф из файла или создать его вручную в графическом интерфейсе. После этого программа выполнит топологическую сортировку и покажет процесс работы алгоритма по шагам.

Во время визуализации будут отображаться:

- текущая обрабатываемая вершина;
- обрабатываемое ребро;
- входящие степени вершин;
- очередь вершин с нулевой входящей степенью;
- текущий результат топологической сортировки;
- текстовое пояснение каждого шага.

Если в графе будет обнаружен цикл, программа сообщит, что топологическая сортировка невозможна.
